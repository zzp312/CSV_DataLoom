## ✨ 核心特点

### 🚀 极致轻量
- **零依赖部署**：无需安装数据库，开箱即用
- **单文件启动**：`python app.py` 一键运行
- **嵌入式引擎**：DuckDB嵌入式数据库，无需额外服务
- **资源占用低**：内存占用小，启动速度快

### 🎨 可视化画布模式
- **点击添加**：点击数据表即可添加到画布
- **智能连线**：点击连接点创建表间关联
- **JOIN类型切换**：INNER/LEFT/RIGHT/FULL JOIN 一键切换
- **字段筛选**：可视化配置筛选条件

### 📝 双模切换
- **画布模式**：可视化构建查询，无需写SQL
- **代码模式**：专业SQL编辑器，支持语法高亮
- **实时同步**：画布操作自动生成SQL语句

### 📊 数据处理
- **多格式导入**：支持 CSV、Excel (.xlsx/.xls)
- **大数据量支持**：轻松处理百万级数据
- **智能类型推断**：自动识别字段类型

## 📦 快速开始

### 环境要求
```
Python 3.10+
```

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动服务
```bash
python app.py
```

### 访问应用
```
http://localhost:8081
```

**无需配置数据库，无需端口映射，启动即可使用！**

## 📖 使用流程

1. **导入数据**
   - 点击左侧导入区域或拖拽文件
   - 支持 CSV 和 Excel 文件
   - 自动解析表结构

2. **构建查询**
   - 切换到"画布模式"
   - 点击左侧数据表添加到画布
   - 悬停查看连接点，点击创建关联

3. **执行查询**
   - 点击"Run Query"按钮
   - 底部面板展示查询结果
   - 支持分页浏览

4. **导出结果**
   - 选择导出格式（CSV/SQL）
   - 一键下载
